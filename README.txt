ТЗ:
1. Используя https://jsonplaceholder.typicode.com/ подтянуть посты
2. Выводить на страницу по 10 постов, реализовать паггинацию
3. Сделать страницу отдельного поста и переход на нее
4. Для минимальной стилизации использовать StyledComponents
5. Для запросов использовать tanstack query
6. Для инициализации проекта использовать Vite
7. Основы стека - React/Ts
В качестве дополнительных требований:
1. Реализовать поиск по постам
2. Следовать принципам FSD

Реализация:
Стэк:
1. Проект Vite + React + TS. Архитектура FSD.
Команда для запуска dev-версии проекта: npm run dev
2. Стили - StyledComponents.
3. Запросы - TanStack Query. Публичное API: https://jsonplaceholder.typicode.com/
Интерфейс:
1. Стартовая страница:
  1.1. Поиск по названию, поиск по тексту: реализованы на основе query-запросов к публичному API.
  NB В рамках тестового задания реализован поиск по одному параметру за раз (либо название, либо текст).
  1.2. Вывод 10 постов на страницу. Переход на страницу отдельного поста - по клику на название поста.
  1.3. Пагинация
2. Страница отдельного поста:
  2.1. Вывод названия, текста, автора поста.
  2.2. Ссылка для возврата на стартовую страницу.
